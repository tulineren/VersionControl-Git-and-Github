## Terminal Kullanımı (Temel Komutlar)

### pwd
Bulunulan dizinin tam yolunu gösterir.

```bash
pwd
/c/Users/TulinEren
```

### ls
Dizindeki dosya ve klasörleri listeler.

```bash
ls
'3D Objects'/
 AppData/
'Application Data'@
 Belgelerim@
'Cisco Packet Tracer 8.2.2'/
 Contacts/
 Cookies@
 Desktop/
 Documents/
 Downloads/
 Favorites/
 Links/
```

### cd
Dizin değiştirmek için kullanılır

```bash
cd Desktop, cd Documents
```

### cd ..
Bir üst dizine geçer.

```bash
cd ..
```

### clear
Terminal ekranını temizler.

```bash
clear
```

### Dosya ve Klasör İşlemleri

### mkdir
Yeni bir klasör oluşturur.

```bash
mkdir GitKursu
```

### touch
Yeni bir dosya oluşturur.

 ```bash
touch not.txt
```

- Terminal kullanarak dosya ve klasör oluşturabiliriz.
